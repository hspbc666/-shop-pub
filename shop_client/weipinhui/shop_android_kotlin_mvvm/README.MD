# 项目功能
1. 显示商品列表
2. 支持搜索商品
3. 支持显示商品信息
4. 支持购买
5. 支持购物车
6. 支持地址管理、订单管理
# 项目技术特色
Kotlin+Retrofit+协程+MVVM
